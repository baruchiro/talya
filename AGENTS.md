## Learned Workspace Facts

- This project is a static Hebrew kids learning app (Aleph-Bet, Colors, Stories) for a 3-year-old; navigation is symbol-only (no text labels) because the user cannot read yet.
- Tile and back icons are images generated from prompts; prompts are documented in docs/icon-prompts.md; replace an icon by regenerating from the prompt and saving as the same filename in img/.
- Home tiles and back control are links containing only the image (no button wrapper).
- Deployment and git workflow are documented in docs/deployment.md; the deploy-github-actions skill describes how to deploy and change the workflow.
